== Description ==
This plugin allows you to add custom JavaScript-based projects to your WordPress site using a simple shortcode. Projects are displayed in a customizable container using JS and CSS.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/js-projects-plugin` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `[js_projects]` shortcode to display the projects on any post or page.

== Frequently Asked Questions ==
= How do I add a project to my page? =
Simply use the `[js_projects]` shortcode wherever you want to display the project.
