# CBSE Class 12 Computer Science Question Bank Plugin

## Overview
This WordPress plugin provides an interactive Class 12 CBSE Computer Science Final Term Question Bank. It allows users to toggle answers for each question in an intuitive and visually appealing format. Perfect for students to practice and test their knowledge interactively.

## Features
- **Interactive UI**: Toggle functionality to show/hide answers for questions.
- **Structured Layout**: Questions organized by sections (MCQs, Very Short Answer, Short Answer, Long Answer, etc.).
- **Easy Integration**: Can be easily embedded into any WordPress post/page using shortcodes.
- **Responsive Design**: Compatible with all device sizes.

## How It Works
This plugin creates an interactive question bank where students can click a button to reveal the correct answer for each question. It leverages HTML, CSS, and JavaScript for smooth transitions and readability.

## Usage

1. **Installation**:
   - Download the plugin files from this repository.
   - Upload the plugin folder to the `/wp-content/plugins/` directory of your WordPress site.
   - Activate the plugin through the 'Plugins' menu in WordPress.

2. **Embedding the Question Bank**:
   - After installing and activating the plugin, you can use the following shortcode in any page or post:
   ```html
   [custom_code]
   <!-- Embed your customized question bank HTML and JavaScript here -->
   [/custom_code]
